## Bugs & Issues Discovered

### 1. card injection R2 URL 상실
- **증상**: 발행 페이지 본문 이미지 `<img src="/images/...">` → HTTP 404
- **원인**: `_publish_hugo()` R2 교체 → index.md 기록 → `inject_cards_chain()`이 DB `draft_md`(원본 `/images/`)로 읽어 덮어쓰기
- **해결**: `published_md` 컬럼 분리 + `_publish_hugo()` 저장 + `inject_into_post()` 전환 + 카드 주입 후 `published_md` 갱신
- **미해결**: 없음

### 2. techpawz R2 404
- **증상**: `img.techpawz.com/images/techpawz/...` → HTTP 404
- **원인**: `img.techpawz.com` → `techpawz-images` 버킷에 바인딩. 코드가 `hotissue-images`에 업로드 → 불일치
- **해결**: `R2_SITE_BUCKETS = {"images/techpawz": "techpawz-images"}` + `_resolve_bucket()`
- **미해결**: 없음

### 3. frontmatter 미해결 마커
- **증상**: republish 시 `<!--todo:image-->` 마커가 unresolved → 본문 이미지 없음
- **원인**: `draft_md`에 마커가 남아있고, `_publish_hugo()`가 이미지 URL을 assets/에 복사하지 못함
- **해결**: `_publish_hugo()`에서 `<!--todo:image/chart-->` → `image_url` DB값으로 자동 해결
- **미해결**: 없음
